General Information about the API

This is a REST(ful) API that uses HTTP requests to receive, send, create and delete data and files.

This API is created for an application called "myFlix" that interacts with a database that stores data about different movies. The users will be able to use this application whenever they like to read information about different movies or update their user information, for instance, their list of favourite movies.

Allow new users to register /users POST A Json object holding data about the user, structured like:
{
"Username" : "bobqueen25",
"Password" : "passwordking25",
"Email" : "bobqueen@gmail.com",
"Birthday" : "19/09/1991"
} 		A verification message that returns a Json object if request is successful, and an error message with a status code if request has failed
Allow existing users to update their information users/update/:Username PUT A Json object holding new user data. Example:
{
"Username": "bobking25", "Password": "passwordking25", "Email": "bobking@gmail.com", "Birthday": 1991-09-19 } 		A verification message that returns a Json object if request is successful, and an error message with a status code if request has failed
Allow users to add a movie to their list of "Favourites" /users/:Username/movies/:MovieID POST None A verification message that returns an error message with a status code if request has failed and a Json object if request is successful, structured like:
{
"_id": "612b8f4c5a1e89e375513217",
"Username": "juliastone",
"Password": "juliastone1",
"Email": "juliastone@gmail.com",
"FavoriteMovies": [
"61293a2a20c07b459bdc67eb",
"6128e72c152a9c8e7ecca52e"
],
"__v": 0
}
